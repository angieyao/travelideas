# Travel Ideas

  從一開始的Javascript "WELCOME!!"到所有分頁的超連結 ，以及時間的顯示都是主頁所必須擁有的。<br>

  不僅介紹了此次網站的內容、結構及規劃概念，更製作了調查表即意見表來更了解大家的想法。 點選每個頁面的名稱即可跳至該頁面。<br>

<h3>/ PICTURES /</h3> 以輪播的方式呈現所有的圖片集。<br>
